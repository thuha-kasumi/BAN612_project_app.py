\# BAN 612: Automated Job Market Analytics App



This Streamlit application is a collaborative tool designed for the BAN 612 Course Project. It allows team members to scrape job postings across multiple industries (Tech, Consulting, Finance, etc.) and centralize the data into a shared Google Sheet.



\## 🚀 Features

\- \*\*Automated Scraping:\*\* Uses `python-jobspy` to search LinkedIn, Indeed, and Glassdoor simultaneously.

\- \*\*Centralized Database:\*\* Automatically saves all entries to a shared Google Sheet.

\- \*\*Member Attribution:\*\* Tracks which team member collected which data points.

\- \*\*Export Options:\*\* Allows downloading individual contributions or the full team dataset as a CSV.



\## 🛠️ Tech Stack

\- \*\*Frontend:\*\* Streamlit

\- \*\*Scraping:\*\* Python-JobSpy

\- \*\*Database:\*\* Google Sheets (via st-gsheets-connection)

\- \*\*Deployment:\*\* Streamlit Community Cloud
